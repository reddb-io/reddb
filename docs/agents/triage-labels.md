# Triage Labels

Use these repository labels for the canonical triage roles:

| Role | Label |
| --- | --- |
| Needs maintainer evaluation | `needs-triage` |
| Waiting on reporter | `question` |
| Ready for an AFK agent | `ready-for-agent` |
| Ready for human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

Use `blocked` when an issue cannot start until another issue lands. Use `release-blocker` only when the issue gates a release.
